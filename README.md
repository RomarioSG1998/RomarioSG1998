## 🌟 Featured Project

### 🎓 [VisioClass](https://github.com/RomarioSG1998) - Student Engagement Monitoring System

> Intelligent real-time monitoring system using **Computer Vision** and **Linear Algebra**

**🔬 Technologies:**
- 🧠 **MediaPipe** - Face Mesh with 468 facial landmarks
- 📊 **PCA** - Principal Component Analysis
- ⚡ **WebSockets** - Real-time communication (50ms)
- 🎯 **FastAPI** - High-performance backend
- ⚛️ **React + TypeScript** - Modern interface

**✨ Features:**
- Attention detection via biometric analysis (EAR - Eye Aspect Ratio)
- Fatigue identification through yawning and posture (MAR - Mouth Aspect Ratio)
- Head pose analysis (Yaw, Pitch, Roll)
- Real-time classification: Attentive, Distracted, Fatigue, Interaction
- Dashboard for teachers with detailed metrics
- Cross-platform desktop apps (Linux/Windows)

**📐 Applied Mathematics:**
- Face representation as matrices R^{468×3}
- Eigenvalues and eigenvectors calculation
- Euclidean norms for facial distances
- Geometric projections for Head Pose
